# Meme Generator

An interactive, dynamic **Meme Generator** web application built using **React** and **Vite**. The application allows users to create custom memes instantly by fetching popular meme templates from a public API and overlaying dynamic top and bottom text fields.

## 🚀 Features

* **Live API Integration:** Fetches a collection of the internet's most popular meme templates asynchronously using React's `useEffect` hook.
* **Dynamic Text Overlay:** Real-time state synchronization updates the top and bottom text over the meme template instantly as the user types.
* **Random Template Generator:** A smooth button click triggers the selection of a brand new, random meme image from the fetched database.
* **Dark Mode Support:** Clean, toggleable dark and light theme options featuring high-contrast UI icons for seamless visibility.
* **Responsive Layout:** Modern, fully responsive UI designed to work beautifully across mobile, tablet, and desktop viewports.

## 💻 Tech Stack

* **Frontend Framework:** React.js (Vite)
* **State Management & Effects:** React Hooks (`useState`, `useEffect`)
* **Data Fetching:** REST API Integration
* **Styling:** CSS3 (Modern layout & responsive design)
* **Deployment:** Live on Netlify
